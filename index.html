Aprendendo HTML

Títulos

As tags h1, h2, h3, h4, h5 e h6 geralmente são utilizadas para títulos de textos e são dispostas em ordem decrescente de tamanho, do maior (h1) ao menor (h6).
Por exemplo:

H1

H2

H3

H4

H5
H6

Parágrafo

A tag (p) é utilizada para fazer parágrafos no site.

Marca Texto

A tag (mark) no HTML é usada para destacar ou marcar texto que deve ser considerado relevante ou
importante em um determinado contexto. Por padrão, o texto envolvido pela tag (mark) é exibido com
um fundo amarelo, simulando o efeito de marca-texto.
Por exemplo:
Este é um texto destacado para chamar a atenção.

A Tag Small

A tag (small) no HTML é usada para indicar texto secundário ou menos importante, geralmente exibindo-o em
um tamanho de fonte menor do que o texto ao redor.
Por exemplo:
Indicar avisos legais ou informações complementares: © Brenda Ramos. Todos os direitos reservados.

Estilos Textuais

Tag i

A tag (i) (de italic) é usada para aplicar estilo itálico ao texto. Além disso, ela indica que o texto tem
um significado especial ou deve ser destacado de alguma forma, como palavras estrangeiras, nomes científicos
ou pensamentos internos.
Por Exemplo:
Eu adoro a expressão carpe diem.

Tag u

A tag (u) (de underline) é usada para aplicar sublinhado ao texto. Em termos semânticos, ela pode ser usada
para indicar texto que requer destaque por algum motivo, como erros de ortografia, nomes próprios em um
documento ou algo incomum.
Por Exemplo:
O nome correto é João da Silva.

Tag strong

A tag (strong) no HTML é usada para indicar importância ou ênfase forte em um texto. Por padrão, o texto
dentro da tag (strong) é exibido em negrito nos navegadores.
Por Exemplo:
É obrigatório preencher todos os campos do formulário.
Obs.: Também é utilizada para exibir o texto em negrito a tag b.

Listas

Listas Ordenadas

As listas ordenadas no HTML são criadas com a tag ol (de ordered list). Elas são usadas para exibir itens em
uma sequência numérica ou ordenada, indicando que a ordem dos itens é importante.
Por Exemplo:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Atributos importantes da tag ol:

1. type: Define o estilo de numeração.
Valores possíves

  • "1": Números (padrão).
  • "A": Letras maiúsculas..
  • "a": Letras minúsculas.
  • "I": Números romanos maiúsculos..
  • "i": Números romanos minúsculos..

Listas Não Ordenadas

As listas não ordenadas no HTML são criadas com a tag ul (unordered list). Elas são usadas para exibir itens
onde a ordem não é importante, geralmente com marcadores (bullets) ao lado dos itens.
Por Exemplo:

  • Item 1
  • Item 2
  • Item 3

Personalização dos marcadores (list-style-type):

O estilo dos marcadores pode ser modificado com o atributo CSS list-style-type.

Exemplos de estilos:
1.Disco (padrão): disc

  • Item 1
  • Item 2

2. Círculo: circle
  • Item 1
  • Item 2

3.Quadrado: square
  • Item 1
  • Item 2

4. Sem marcadores: none
  • Item 1
  • Item 2

Tag li

A tag li (de list item) no HTML é usada para representar itens individuais dentro de uma lista. Ela pode ser
usada tanto em listas ordenadas (ol) quanto em listas não ordenadas (ul).

Exemplo com lista não ordenada:

(ul)
(li)Item 1(li)
(li)Item 2(li)
(li)Item 3(li)
(ul):
  • Item 1
  • Item 2
  • Item 3

Exemplo com lista ordenada:

(ol)
(li)Passo 1(li)
(li)Passo 2(li)
(li)Passo 3(li)
(ol):
  1. Passo 1
  2. Passo 2
  3. Passo 3

Hiperlink

A tag a no HTML é usada para criar hiperlinks, permitindo que o usuário navegue entre páginas da web, seções de
um documento ou até mesmo inicie downloads. A a significa anchor (âncora).
Por Exemplo:
(a href="https://www.exemplo.com")Clique aqui para acessar o site (a)
Clique aqui para acessar o site

Atributos principais da tag (a):

1. href (Hyperlink Reference):
Define o destino do link. Pode ser uma URL externa, um caminho interno ou um identificador de âncora.
- URL externa:
(a href="https://www.google.com")Google(a)
Resultado:
Google
- Caminho interno:
(a href="/sobre.html")Sobre nós(/a)
Resultado:
Sobre nós
- Âncora interna (mesma página):
(a href="#seção1")Ir para a seção 1(/a)
Resultado:
Ir para a seção 1

2.target:
Define como o link será aberto.
_self (padrão): Abre no mesmo navegador.
_blank: Abre em uma nova aba ou janela.
_parent: Abre no quadro pai (em contexto de frames).
_top: Abre na janela principal, ignorando frames.
(a href="https://www.exemplo.com" target="_blank")Abrir em nova aba(a)

3. rel:
Define a relação entre o documento atual e o link. Alguns valores comuns são:
- nofollow: Indica aos mecanismos de busca para não seguirem o link.
- noopener: Melhora a segurança ao abrir links em nova aba.
(a href="https://www.exemplo.com" target="_blank" rel="noopener")Link seguro(/a)

4. download:
- Permite que o link seja usado para fazer o download de um arquivo.
(a href="documento.pdf" download)Baixar PDF(/a)

A Tag hr

A tag hr no HTML é usada para inserir uma linha horizontal que serve para separar conteúdo em
uma página web. Ela é uma tag autocontida (não requer uma tag de fechamento) e, além de ter um
papel visual, possui significado semântico para indicar uma mudança de tópico ou uma divisão temática.
Por Exemplo:
(p)Seção(p)
(hr)
(p)Seção 2(p)

Seção 1

Seção 2

A Tag sub

A tag sub no HTML é usada para exibir texto em subscrito, ou seja, o texto é posicionado abaixo da linha de
base do texto normal. Ela é comumente utilizada em expressões químicas, fórmulas matemáticas, e outras situações
em que a formatação subscrita é necessária.
Exemplo:
(p)H(sub)2(/sub)O(/p)

H2O

Resultado:
- H₂O (onde o número "2" está abaixo da linha de base)

A Tag sup

A tag sup no HTML é usada para exibir texto em sobrescrito, ou seja, o texto é posicionado acima da linha de
base do texto normal. Ela é comumente utilizada em expressões matemáticas, fórmulas químicas, potências,
expoentes e outras situações em que a formatação sobrescrita é necessária.
Exemplo:
(p)E = mc(sup)2(/sup)(/p)

E = mc2

Resultado:
- E = mc² (onde o número "2" está acima da linha de base)

Citação

A tag blockquote no HTML é usada para representar citações em bloco, ou seja, trechos de texto retirados de
uma fonte externa. Ela é normalmente utilizada para destacar uma citação longa ou importante, e geralmente é
exibida com uma margem à esquerda para indicar que se trata de uma citação.
Exemplo:
(blockquote)
"A vida é o que acontece enquanto estamos ocupados fazendo outros planos."
(footer)- John Lennon(/footer)
(/blockquote)

"A vida é o que acontece enquanto estamos ocupados fazendo outros planos."
- John Lennon

Atributos principais:

1. cite:
O atributo cite é utilizado para fornecer a URL ou referência de onde a citação foi retirada. Isso ajuda a
dar contexto à citação e pode ser útil para fins de credibilidade.
Exemplo:
(blockquote cite="https://www.exemplo.com/autor")
"A persistência é o caminho do êxito."
(footer)- Charles Chaplin(/footer)
(/blockquote)

"A persistência é o caminho do êxito."
- Charles Chaplin
2. footer: Dentro da tag (blockquote), é comum usar a tag (footer) para fornecer informações adicionais sobre a
citação, como o autor, a fonte ou a data.
